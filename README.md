# CNN — apoyo visual para la presentación

Presentación interactiva del funcionamiento de una red convolucional (CNN),
apoyo visual para la defensa del Proyecto Final I302 (Caminoa · Krinisky,
Universidad de San Andrés).

Presentación: [`cnn_animacion.html`](cnn_animacion.html)

Se abre en cualquier navegador. Avanzá con las flechas **← →**, la barra
espaciadora o un **tap**. En iPad: abrila en Safari y *Compartir → Agregar a
inicio* para tenerla en pantalla completa.

> Publicado con GitHub Pages solo como apoyo visual.
